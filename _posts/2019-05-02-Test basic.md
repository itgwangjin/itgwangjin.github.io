# 1. What is A/B Test
## 1) 의미
1. 시스템 방문자를 임의로 두 집단으로 나눈다.
2. 한 집단에게는 기존 사이트를 보여준다
3. 다른 집단에게는 새로운 사이트를 보여준다.
4. 두 집단 중 어떠한 집단이 더 높은 만조
